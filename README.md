# AI智能零售柜竞品分析系统 - 用户手册

## 系统简介

AI智能零售柜竞品分析系统是一款专为零售柜行业设计的数据分析工具，帮助用户快速分析和对比不同品牌的智能零售柜产品，为采购决策提供数据支持。

### 核心功能
- **产品管理**：添加、编辑、删除和导入产品信息
- **竞品分析**：多维度对比不同产品的性能和价格
- **数据可视化**：通过图表直观展示分析结果
- **AI分析**：使用AI技术生成深度分析报告
- **报告生成**：自动生成专业分析报告

## 快速开始

### 1. 系统登录

1. **访问系统**
   - 打开浏览器，输入系统地址：`http://localhost:8080`
   - 点击右上角的用户图标，选择"登录"

2. **登录账号**
   - **管理员账号**：用户名 `admin`，密码 `admin123`
   - **测试账号**：用户名 `test`，密码 `test123`
   - 输入账号密码后点击"登录"按钮

3. **注册新账号**
   - 如果没有账号，点击"立即注册"
   - 填写用户名、邮箱和密码
   - 点击"注册"按钮完成注册

### 2. 系统导航

登录后，您将看到系统的主界面，包括以下部分：

- **顶部导航栏**：显示当前页面标题、搜索框和用户菜单
- **左侧菜单**：包含系统的主要功能模块
- **内容区域**：显示当前功能模块的详细内容

## 功能使用指南

### 1. 产品管理

#### 添加产品
1. 点击左侧菜单的"产品管理"
2. 点击"添加产品"按钮
3. 填写产品信息：
   - **品牌**：产品品牌名称
   - **型号**：产品型号（唯一标识）
   - **名称**：产品完整名称
   - **价格**：产品价格
   - **颜色**：产品品牌颜色（十六进制格式）
   - **规格参数**：填写产品的详细规格
   - **性能指标**：填写产品的性能数据
4. 点击"保存"按钮

#### 导入产品
1. 点击"导入Excel"按钮
2. 在弹出的模态框中，点击"浏览文件"
3. 选择包含产品信息的Excel文件（支持.xlsx和.csv格式）
4. 系统会自动解析文件并显示在表格中
5. 可以在表格中编辑产品信息
6. 点击"确认导入"按钮完成导入

#### 编辑产品
1. 在产品列表中找到要编辑的产品
2. 点击产品卡片上的"编辑"按钮
3. 修改产品信息
4. 点击"保存"按钮

#### 删除产品
1. 在产品列表中找到要删除的产品
2. 点击产品卡片上的"删除"按钮
3. 在确认对话框中点击"确认"按钮

### 2. 竞品分析

#### 产品对比
1. 点击左侧菜单的"竞品分析"
2. 在"产品对比"部分，选择要对比的产品（最多4个）
3. 系统会自动生成对比表格和图表
4. 可以切换不同的对比维度：规格、性能、价格

#### 性能分析
1. 在"性能分析"部分，选择要分析的产品
2. 系统会生成性能指标雷达图
3. 可以查看不同性能指标的详细数据

#### 价格分析
1. 在"价格分析"部分，系统会自动生成价格分布图表
2. 可以查看不同品牌的价格区间
3. 可以查看价格与性能的关系分析

#### 品牌分析
1. 在"品牌分析"部分，选择要分析的品牌
2. 系统会生成品牌对比图表
3. 可以查看不同品牌的产品特点

### 3. 报告生成

#### AI分析
1. 点击左侧菜单的"报告生成"
2. 在"AI分析"部分，选择要分析的产品（最多3个）
3. 点击"生成AI分析"按钮
4. 系统会自动生成深度分析报告，包括：
   - 产品综合评分
   - 优势劣势分析
   - 市场定位建议
   - 采购决策建议

#### 导出报告
1. 在"历史报告"部分，找到要导出的报告
2. 点击报告卡片上的"导出"按钮
3. 选择导出格式（Excel或PDF）
4. 系统会自动生成并下载报告文件

#### 管理报告
1. **编辑报告**：点击报告卡片上的"编辑"按钮
2. **删除报告**：点击报告卡片上的"删除"按钮
3. **查看报告**：点击报告卡片查看详细内容

### 4. 数据分析

#### 销售趋势分析
1. 点击左侧菜单的"数据分析"
2. 在"销售趋势"部分，选择时间范围
3. 系统会生成销售趋势图表
4. 可以查看不同产品的销售情况

#### 市场份额分析
1. 在"市场份额"部分，选择分析维度
2. 系统会生成市场份额饼图
3. 可以查看不同品牌的市场占比

#### 客户满意度分析
1. 在"客户满意度"部分，系统会自动生成满意度图表
2. 可以查看不同产品的客户评价

## 常见问题

### 1. 登录失败
- **原因**：用户名或密码错误
- **解决方法**：检查输入的用户名和密码是否正确

### 2. 无法添加产品
- **原因**：必填字段未填写或型号重复
- **解决方法**：检查所有必填字段并确保型号唯一

### 3. Excel导入失败
- **原因**：文件格式不正确或数据格式错误
- **解决方法**：确保文件是.xlsx或.csv格式，并检查数据格式

### 4. 报告生成失败
- **原因**：选择的产品数量过多或系统资源不足
- **解决方法**：减少选择的产品数量或稍后再试

### 5. 图表显示异常
- **原因**：浏览器兼容性问题或数据异常
- **解决方法**：使用现代浏览器（如Chrome、Firefox）并检查数据是否正确

## 性能提示

- **大数据量**：当产品数量超过100个时，建议使用筛选和搜索功能
- **图表渲染**：当同时显示多个图表时，可能会影响性能
- **AI分析**：AI分析功能需要一定时间处理，请耐心等待
