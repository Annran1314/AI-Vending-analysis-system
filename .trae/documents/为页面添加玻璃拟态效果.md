# 为页面添加玻璃拟态效果

## 1. 分析当前页面结构
- **页面主题**：深色主题，背景色为 `bg-dark-300`
- **主要组件**：
  - 侧边导航栏 (`aside`)
  - 顶部导航栏 (`header`)
  - 移动端导航栏
  - 卡片式组件 (`.card`)
  - 模态框组件
  - 按钮和表单元素

## 2. 实现玻璃拟态效果的方案

### 2.1 在 Tailwind 配置中添加玻璃拟态工具类
- 添加 `.glass-effect` 工具类，包含以下属性：
  - 半透明背景 (`background-color: rgba(...)`)
  - 模糊效果 (`backdrop-filter: blur()`)
  - 细微边框 (`border`)
  - 轻微阴影 (`box-shadow`)

### 2.2 修改主要组件

#### 2.2.1 导航栏组件
- 侧边导航栏：添加玻璃拟态效果
- 顶部导航栏：添加玻璃拟态效果
- 移动端导航栏：添加玻璃拟态效果

#### 2.2.2 卡片组件
- 概览卡片：添加玻璃拟态效果
- 图表卡片：添加玻璃拟态效果
- 竞品卡片：添加玻璃拟态效果

#### 2.2.3 模态框组件
- 添加竞品模态框：添加玻璃拟态效果
- 分析进度模态框：添加玻璃拟态效果
- Excel 导入模态框：添加玻璃拟态效果

#### 2.2.4 其他组件
- 按钮：保持扁平化设计，不添加玻璃效果
- 表单元素：保持扁平化设计，不添加玻璃效果
- 下拉菜单：添加玻璃拟态效果

### 2.3 调整背景和层次
- 确保背景有足够的对比度，使玻璃拟态效果明显
- 调整组件的 `z-index`，确保玻璃效果的层次感
- 为玻璃组件添加适当的悬停效果

## 3. 具体实现步骤

### 3.1 修改 Tailwind 配置
在 `<script>` 标签中的 `tailwind.config` 对象中添加玻璃拟态工具类。

### 3.2 更新 CSS 样式
在 `<style type="text/tailwindcss">` 标签中添加玻璃拟态的具体样式定义。

### 3.3 修改 HTML 组件
为需要添加玻璃拟态效果的组件添加 `.glass-effect` 类。

### 3.4 测试和调整
- 确保玻璃拟态效果在不同屏幕尺寸下正常显示
- 确保玻璃拟态效果与现有扁平化设计风格相融合
- 确保页面的可读性和可用性不受影响

## 4. 预期效果
- 页面保持扁平化设计的整体风格
- 主要组件添加了玻璃拟态效果，增加了现代感和层次感
- 玻璃拟态效果与深色主题相得益彰
- 页面在视觉上更加美观和专业
